# seller_tax_no → tin 修改详情

## 修改说明

将字段key从 `seller_tax_no` 统一修改为 `tin` (Tax Identification Number - 纳税人识别号)

**修改原因**:
- TIN是国际标准的纳税人识别号缩写
- 更简洁且语义明确
- 统一了"销售方识别号"、"纳税人识别号"、"销售方纳税人识别号"等多种表述

## 影响范围

### 查验接口

- **类型**: string
- **最大长度**: 20
- **说明**: 用于标识销售方/纳税主体的税号
